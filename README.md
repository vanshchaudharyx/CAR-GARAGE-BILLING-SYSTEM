# CAR-GARAGE-BILLING-SYSTEM
A simple Command Line Interface (CLI) based Car Garage Billing System built using Python and MySQL.<br>

This project helps manage:<br>

Customers<br>
Customer vehicles<br>
Garage services<br>
Bill generation<br>
Bill records<br>

The project uses Python classes and methods to perform different operations and MySQL to store the data permanently.<br>

<hr>
Features<br>
Add a new customer<br>
View all customers<br>
Add a vehicle for an existing customer<br>
View all vehicles<br>
Generate a service bill<br>
View all generated bills<br>
Store data permanently in MySQL<br>
Use foreign key relationships between tables<br>
Automatically delete related records using ON DELETE CASCADE<br>

<hr>
Technologies Used<br>
Python<br>
MySQL<br>
MySQL Connector for Python<br>
Object-Oriented Programming (OOP)<br>
SQL Joins<br>
Command Line Interface (CLI)<br>
